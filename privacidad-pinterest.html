Política de Privacidad – Integración con la API de Pinterest

Última actualización: junio de 2026

Esta política describe cómo Arquitecto Digital ("nosotros") trata los datos en relación con el uso de la API de Pinterest. Complementa nuestra Política de Privacidad general.

Quiénes somos

Arquitecto Digital es un pequeño negocio con sede en El Puerto de Santa María (Cádiz, España). Sitio web: arquitectodigital.io.

Uso de la API de Pinterest

Nuestra aplicación utiliza la API oficial de Pinterest (v5) con un único propósito: publicar y programar nuestro propio contenido visual (imágenes y enlaces a nuestra web y productos) en nuestra propia cuenta de empresa de Pinterest.

La aplicación no accede, gestiona ni publica en cuentas de terceros. No solicitamos credenciales de otros usuarios de Pinterest.

Datos que tratamos

Para operar contra la API de Pinterest tratamos exclusivamente:

  • El token de acceso OAuth de nuestra propia cuenta de empresa.
  • Los identificadores de nuestros tableros y pines.
  • El contenido (imágenes, títulos, descripciones y enlaces) que nosotros mismos publicamos.

No recopilamos datos personales de otros usuarios de Pinterest, ni perfiles, ni seguidores, ni datos de navegación de terceros.

Compartición de datos

No vendemos, alquilamos ni compartimos con terceros ningún dato obtenido a través de la API de Pinterest. Los tokens de acceso se almacenan de forma segura y se utilizan únicamente para autenticar nuestras propias solicitudes a la API.

Conservación y eliminación

Los tokens de acceso se conservan mientras la integración esté activa. Puedes solicitar la revocación del acceso en cualquier momento; los tokens se eliminan de nuestros sistemas tras la revocación.

Cumplimiento

El uso de la API de Pinterest se rige por los términos y políticas de desarrollador de Pinterest, además del Reglamento General de Protección de Datos (RGPD) aplicable en la Unión Europea.

Contacto

Para cualquier consulta sobre esta política o sobre el tratamiento de datos: info@arquitectodigital.io.
